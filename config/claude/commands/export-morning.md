# 朝の日報生成 (export-morning)

`~/.claude/nippo/tasks.md` を元に、本日の朝日報を生成します。

## 手順

1. `~/.claude/nippo/tasks.md` を読み込む（存在しない場合は空のタスクリストとして扱う）
2. 今日の日付（YYYY-MM-DD）を取得する
3. 朝日報マークダウンを生成する
4. `~/.claude/nippo/reports/YYYY-MM-DD-morning.md` に保存する
5. チャットにマークダウン内容を表示する

## 朝日報のフォーマット

```markdown
🌅 開始
【本日の予定】

■ 今日やること
{未完了タスクを優先度順に列挙。プロジェクト別にグループ化}
- [ ] {タスク名}

■ 完了ライン
{未完了タスクの数・傾向から、今日すべきことを1〜2文で提案}
- ○○が動く状態
- ○○まで実装完了
```

## 各セクションの埋め方

- **今日やること**：tasks.md の未完了タスクを優先度順に列挙。プロジェクト名が付いているタスクはプロジェクト別にグループ化する（例：`プロジェクト: ADeT-AI` のタスクをまとめて並べる）。プロジェクト不明のタスクは最後にまとめる
- **完了ライン**：今日中に達成すべき具体的な状態を1〜2行で記述。タスク名をそのまま並べるのではなく「○○が動く状態」「○○まで実装完了」のように成果物・状態ベースで書く

## タスクの優先度判定基準

- **優先度高**：追加日が3日以上前のタスク、または名前に「緊急」「urgent」「fix」「bug」を含むもの
- **優先度中**：当日・前日追加のタスク
- **持ち越し**：追加から7日以上経過しているタスク

## 応答

チャットには以下を表示する：
1. 生成した朝日報の全文（マークダウンレンダリング）
2. 保存先パス

例：
```
📋 朝の日報を生成しました

[日報内容]

💾 保存先: ~/.claude/nippo/reports/2026-03-18-morning.md
```
