# タスク一覧表示 (nippo:list)

`~/.claude/nippo/tasks.md` を読み込み、現在の未完了タスク一覧を表示します。

## 手順

1. `~/.claude/nippo/tasks.md` を読み込む
2. ファイルが存在しない場合、その旨を伝えて終了する
3. 引数がある場合、その条件でフィルタリングする
4. タスク一覧を整形して表示する

## 表示フォーマット

以下の形式で表示する：

```
📋 タスク一覧

## 未完了タスク（N件）

{プロジェクト別にグループ化して表示}
### プロジェクト: {プロジェクト名}
- {タスク名} | 追加: YYYY-MM-DD

### その他
- {タスク名} | 追加: YYYY-MM-DD

📁 ~/.claude/nippo/tasks.md
```

## フィルタリング

引数が指定された場合：
- プロジェクト名が指定された場合：そのプロジェクトのタスクのみ表示

## タスクが0件の場合

```
📋 タスク一覧

未完了タスクはありません。

/nippo:update でタスクを追加できます。
```
