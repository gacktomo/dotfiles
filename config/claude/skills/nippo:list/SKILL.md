# タスク一覧表示 (nippo:list)

当日の朝日報から「今日やること」リストと「作業進捗メモ」を読み込み、現在の状況を表示します。

## 手順

1. 今日の日付（YYYY-MM-DD）を取得する
2. `~/.claude/nippo/reports/YYYY-MM-DD-morning.md` を読み込む
3. ファイルが存在しない場合、朝日報がまだ作成されていない旨を伝え `/nippo:morning` を案内して終了する
4. `■ 今日やること` セクションと `■ 作業進捗メモ` セクションを抽出して表示する

## 表示フォーマット

```
📋 本日のタスク状況（YYYY-MM-DD）

■ 今日やること
{朝日報の「今日やること」セクションをそのまま表示}

■ 作業進捗メモ
{朝日報の「作業進捗メモ」セクションをそのまま表示。セクションが空または存在しない場合は「（まだメモなし）」と表示}

📁 ~/.claude/nippo/reports/YYYY-MM-DD-morning.md
```

## 朝日報が存在しない場合

```
⚠️ 本日（YYYY-MM-DD）の朝日報がまだ作成されていません。

/nippo:morning で朝日報を作成してください。
```
