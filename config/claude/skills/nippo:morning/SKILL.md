# 朝の日報生成 (nippo:morning)

前日の夕日報の `■ 明日の予定` セクションを参照して、本日の朝日報を生成します。

## 手順

1. 今日の日付（YYYY-MM-DD）を取得する
2. 前日の日付（YYYY-MM-DD）を計算する
3. `~/.claude/nippo/reports/YYYY-MM-DD-evening.md`（前日）を読み込む
4. 前日夕日報の `■ 明日の予定` セクションを抽出して「今日やること」に使用する
5. 前日夕日報が存在しない場合は、ユーザーに今日やることを確認する
6. 朝日報マークダウンを生成する
7. `~/.claude/nippo/reports/YYYY-MM-DD-morning.md`（今日）に保存する
8. チャットにマークダウン内容を表示する

## 朝日報のフォーマット

```markdown
🌅 開始
【本日の予定】

■ 今日やること
{前日夕日報の「明日の予定」を引き継いで番号付きリストで記述}
1. {タスク名}

■ 作業進捗メモ


■ 完了ライン
{今日やることの数・内容から、今日すべき達成状態を1〜2文で提案。通し番号はタスクと一致させる}
1. ○○が動く状態
2. ○○まで実装完了
```

## 前日夕日報が存在しない場合

ユーザーに「今日やることを教えてください」と確認し、その内容で `■ 今日やること` を作成する。

## 応答

チャットには以下を表示する：
1. 生成した朝日報の全文（マークダウンレンダリング）
2. 保存先パス

例：
```
📋 朝の日報を生成しました

[日報内容]

💾 保存先: ~/.claude/nippo/reports/2026-03-18-morning.md
```
