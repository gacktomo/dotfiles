# タスク追加 (add-target)

タスクを `~/.claude/nippo/tasks.md` に追加します。

## 手順

1. `~/.claude/nippo/tasks.md` を読み込む（存在しない場合は後で作成）
2. 引数からタスク名を取得する。引数がない場合はユーザーに確認する
3. 現在の日付（YYYY-MM-DD形式）を取得する
4. タスクを追記する
5. 追加完了をユーザーに伝える

## tasks.md のフォーマット

ファイルが存在しない場合、以下の初期構造で作成する：

```markdown
# 日報タスクリスト

## 未完了タスク

## 完了タスク

## 作業ログ
```

タスク追加時は「未完了タスク」セクションに以下の形式で追記する：

```
- [ ] {タスク名} | 追加: {YYYY-MM-DD}
```

プロジェクトが会話コンテキストから判別できる場合（例：現在のリポジトリ名）は以下の形式：

```
- [ ] {タスク名} | プロジェクト: {プロジェクト名} | 追加: {YYYY-MM-DD}
```

## 応答

追加後、以下を表示する：
- 追加したタスクの内容
- 現在の未完了タスク数
- tasks.md のパス

例：
```
✅ タスクを追加しました：
  [ ] search-spec ツールの実装 | プロジェクト: ADeT-AI | 追加: 2026-03-18

📋 未完了タスク: 3件
📁 ~/.claude/nippo/tasks.md
```
